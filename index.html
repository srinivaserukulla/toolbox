ToolBox

A jQuery plugin that creates Application style ToolBox.

Notepad is a handy app when you want to view or make simple edits to text files (usually with a .txt extension). If you'd like to do something more advanced—use font colors, insert pictures, or type in other languages—you might want to use WordPad.

Usage

Include the plugin javascript file along with jquery

<script src="jquery.min.js"></script>
<script src="toolbox.min.js"></script>

Include the css files

<link href="toolbox.css" rel="stylesheet" />
<link href="briskcss.css" rel="stylesheet" />

Define your toolbox HTML

<div class="toolbox">
    <div class="title">
        <div class="title_btn">
            <button id="toolbox-close-btn"> ... </button>
        </div>
        <div class="title_name"> ... &/div>
    </div>
<div class="clearfix"></div>
   
    <div class="toolbox_menu">
      <ul>
        <li title=" ... "> 
            <span class="briskicon briskicon-calendar"> ... </span>
            <p> ... </p>
        </li>
      </ul>
    </div>

    <div class="toolbox_menu">
      <ul>
        <li title=" ... "> 
            <span class="briskicon briskicon-calendar"> ... </span>
            <p> ... </p>
        </li>
      </ul>
    </div>

    <div class="clearfix"></div>
</div>
