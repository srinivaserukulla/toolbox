Download

Features

  • In This toolbox is fixed size.
  • Toolbox icon use image also.
  • Now Single line and with out text
  • In tool mouse over highlight tool name is visible
  • It's Now six Colors is available.

$("#toolbox-close-btn").click(function () {
	$("#toolbox-close-btn span").toggleClass("briskicon briskicon-backward2").toggleClass("briskicon briskicon-forward5");
	$(".toolbox p").toggle("slide");
	$('.toolbox li').removeClass('active');
});

$(".toolbox li").click(function () {
	$('.toolbox li').removeClass('active');
	$(this).addClass("active");
});

Usage

Include the plugin javascript file along with jquery

<script src="jquery.min.js"></script>
<script src="toolbox.min.js"></script>

Include the css files

<link href="toolbox.css" rel="stylesheet" />
<link href="briskcss.css" rel="stylesheet" />

Define your toolbox HTML

<div class="toolbox">
    <div class="title">
        <div class="title_btn">
            <button id="toolbox-close-btn"> ... </button>
        </div>
        <div class="title_name"> ... &/div>
    </div>
<div class="clearfix"></div>
   
    <div class="toolbox_menu">
      <ul>
        <li title=" ... "> 
            <span class="briskicon briskicon-calendar"> ... </span>
            <p> ... </p>
        </li>
      </ul>
    </div>

    <div class="toolbox_menu">
      <ul>
        <li title=" ... "> 
            <span class="briskicon briskicon-calendar"> ... </span>
            <p> ... </p>
        </li>
      </ul>
    </div>

    <div class="clearfix"></div>
</div>
